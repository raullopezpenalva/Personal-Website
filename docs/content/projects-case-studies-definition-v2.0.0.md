# Projects / Case Studies Definition — v2.0.0

## Overview

Este documento define la nueva estructura de proyectos / case studies para la personal website `v2.0.0`.

El objetivo es transformar la página Projects desde una lista de proyectos técnicos o DevOps hacia un portfolio profesional que muestre la experiencia completa:

> Sonido · AV · Eventos · Corporate AV · Infraestructura · Software

La idea no es documentar cada bolo o cada trabajo aislado, sino agrupar la trayectoria en grandes bloques profesionales que expliquen capacidades reales, contexto técnico y valor aportado.

El portfolio debe demostrar que el perfil no es solo software, ni solo sonido, ni solo DevOps.

Debe comunicar:

> Experiencia real en AV, sonido, eventos, producción técnica, entornos corporativos, redes, infraestructura y desarrollo de herramientas propias.

---

# 1. Estructura final recomendada

La página Projects / Case Studies se organizará inicialmente en seis grandes case studies:

```txt
1. Sound Engineering para eventos y espectáculos
2. Soporte técnico de sonido en gira con Ominira
3. Sonido directo y postproducción de audio para audiovisual
4. AV corporativo, conferencias y eventos híbridos
5. Homelab e infraestructura self-hosted
6. Web personal y proyectos software
```

Versión inglesa:

```txt
1. Sound Engineering for Live Events and Shows
2. Touring Sound Support with Ominira
3. Location Sound and Audio Postproduction for AV
4. Corporate AV, Conferences and Hybrid Events
5. Homelab and Self-hosted Infrastructure
6. Personal Website and Software Projects
```

Esta estructura evita convertir el portfolio en una lista infinita de trabajos y permite presentar cada bloque como una capacidad profesional consolidada.

# 2. Case Study 1 — Sound Engineering para eventos y espectáculos

## ES — Título

```txt
Sound Engineering para eventos y espectáculos
```

## EN — Title

```txt
Sound Engineering for Live Events and Shows
```

## Categoría

```txt
Sound Engineering / Live Events
```

## Qué agrupa

Este case study agrupa la experiencia como técnico de sonido en eventos, espectáculos, festivales, conciertos, bolos sueltos y trabajos realizados dentro de empresas audiovisuales como Oxigen.

Incluye también trabajos relacionados con:

```txt
FOH
monitores
pruebas de sonido
ruteo de señal
operación de consolas
producción técnica de espectáculos
soporte a producción
técnico de escenario
eventos culturales
festivales
sistemas d&b / DAS
calibración y ajuste básico de sistemas
```

## Contexto

Durante varios años he trabajado en entornos reales de sonido directo, tanto como freelance como dentro de estructuras de producción técnica.

En este bloque entran trabajos de alta rotación de eventos, operación de sonido, coordinación con producción, trato con artistas y adaptación a espacios con condiciones técnicas muy variables.

Uno de los contextos más relevantes fue el trabajo en Oxigen Sonido e Iluminación, donde participé en producción técnica de espectáculos, presupuestos, coordinación de eventos, operación FOH y monitores, además de trabajos relacionados con sistemas de sonido profesionales.

## Reto

El reto principal en este tipo de trabajo es operar bajo presión real:

```txt
tiempos de montaje ajustados
riders incompletos o cambiantes
necesidades de artistas
limitaciones del espacio
equipos diferentes en cada evento
coordinación con producción
incidencias de señal
pruebas de sonido rápidas
```

No se trata solo de mezclar. Se trata de entender el flujo completo del evento y mantener el sistema operativo desde el montaje hasta el final del show.

## Rol asumido

Los roles asumidos dentro de este bloque incluyen:

```txt
FOH Sound Engineer
Monitor Engineer
Stage Technician
Technical Production Support
Sound System Support
Event Technical Operator
```

## Enfoque técnico

El enfoque se basa en:

```txt
revisar necesidades técnicas antes del evento
entender el flujo de señal
verificar patch y ruteo
organizar prueba de sonido
mantener comunicación clara con artistas y producción
operar el show con criterio técnico y musical
resolver incidencias sin romper el ritmo del evento
documentar aprendizajes y puntos críticos
```

## Herramientas / sistemas

```txt
consolas digitales
sistemas PA
monitores
microfonía
ruteo de señal
Dante cuando aplica
sistemas d&b audiotechnik
sistemas DAS Audio
checklists
riders
input lists
stage plots
```

## Resultado

Este bloque demuestra experiencia real y acumulada en operación de sonido para eventos y espectáculos.

La experiencia no se limita a un único entorno, sino que incluye múltiples espacios, clientes, formatos, artistas y equipos técnicos.

## Qué demuestra

```txt
Capacidad para trabajar bajo presión.
Criterio de sonido en entornos reales.
Comprensión del flujo completo de señal.
Comunicación con producción, artistas y equipo técnico.
Fiabilidad durante montaje, prueba y show.
Capacidad para adaptarse a espacios y equipos distintos.
```

## Resumen para card

```txt
Experiencia en mezcla FOH, monitores, pruebas de sonido, coordinación técnica y operación de eventos en entornos reales de producción.
```

## Tags sugeridos

```txt

Eventos

```

# 3. Case Study 2 — Soporte técnico de sonido en gira con Ominira

## ES — Título

```txt
Soporte técnico de sonido en gira con Ominira
```

## EN — Title

```txt
Touring Sound Support with Ominira
```

## Categoría

```txt
Sound Engineering / Touring
```

## Qué agrupa

Este case study recoge el trabajo realizado con el grupo musical Ominira en más de 50 conciertos, con giras por Italia y norte de España.

## Contexto

Trabajar con una banda de forma continuada es distinto a realizar bolos aislados.

La relación técnica se construye con el tiempo: se entiende mejor el show, las necesidades de los músicos, las dinámicas de escenario, los puntos críticos y la forma de adaptar el concierto a espacios distintos.

## Reto

El reto principal fue mantener una experiencia técnica y sonora coherente en contextos muy variables:

```txt
diferentes salas
diferentes sistemas de sonido
diferentes equipos locales
tiempos de prueba cambiantes
condiciones técnicas variables
necesidades específicas de la banda
adaptación constante del show
```

## Rol asumido

```txt
Sound Engineer
Band Technician
Live Sound Support
Touring Technical Support
```

## Enfoque técnico

El enfoque consistía en:

```txt
entender las necesidades recurrentes de la banda
adaptar el show a cada espacio
coordinarse con técnicos locales
preparar pruebas de sonido eficientes
resolver problemas de señal o monitorización
mantener comunicación directa con músicos
asegurar continuidad técnica durante la gira
```

## Herramientas / sistemas

```txt
consolas de directo
microfonía
monitores
sistemas PA locales
riders
input lists
stage plots
adaptación a equipamiento de sala
```

## Resultado

La experiencia con Ominira demuestra continuidad profesional, confianza con una banda y capacidad para trabajar en gira adaptando el show a entornos distintos.

No es solo experiencia técnica puntual, sino trabajo sostenido con una formación musical real.

## Qué demuestra

```txt
Continuidad profesional.
Relación técnica con artistas.
Adaptación a venues diferentes.
Capacidad de resolver problemas en gira.
Comprensión del show desde dentro.
Fiabilidad en contextos cambiantes.
```

## Resumen para card

```txt
Soporte técnico de sonido en más de 50 conciertos con una banda, adaptando el show a diferentes espacios, equipos y condiciones reales de gira.
```

# 4. Case Study 3 — Sonido directo y postproducción de audio para audiovisual

## ES — Título

```txt
Sonido directo y postproducción de audio para audiovisual
```

## EN — Title

```txt
Location Sound and Audio Postproduction for AV
```

## Categoría

```txt
Location Sound / Audio Postproduction
```

## Qué agrupa

Este case study agrupa trabajos de sonido en rodaje, microfonía, boom, edición de diálogos, limpieza de audio, diseño sonoro, mezcla y normalizado para proyectos audiovisuales.

Incluye experiencia en:

```txt
rodajes
cortometrajes
película
cápsulas publicitarias
spots
contenido corporativo
captación de diálogos
edición de diálogos
postproducción de sonido
```

Entre los trabajos destacados se incluye la participación como microfonista y editor de diálogos en la película `El fred que crema`, además de trabajos de grabación, edición, diseño sonoro, mezcla y normalizado para proyectos publicitarios y corporativos.

## Contexto

El sonido audiovisual tiene una lógica distinta al sonido directo de eventos.

En rodaje, el objetivo es capturar sonido claro, útil y coherente con la imagen. En postproducción, el reto es limpiar, editar, organizar y preparar el audio para que funcione dentro del producto final.

## Reto

Los retos habituales en este bloque incluyen:

```txt
captar diálogo claro en localizaciones reales
minimizar ruido de fondo
gestionar microfonía lavalier
operar boom
captar ambientes
mantener continuidad sonora
limpiar diálogos
editar tomas
integrar sonido dentro del flujo de postproducción
```

## Rol asumido

```txt
Location Sound Recordist
Boom Operator
Dialogue Editor
Audio Postproduction Technician
Sound Editor
```

## Enfoque técnico

El enfoque se basa en entender el flujo completo de audio audiovisual:

```txt
captación en set
microfonía adecuada
claridad de diálogo
control de ruido
organización de material
edición de diálogos
limpieza y preparación
diseño sonoro
mezcla
normalización
entrega final
```

## Herramientas / sistemas

```txt
grabadores de campo
boom
micrófonos shotgun
lavaliers
sistemas inalámbricos
edición de audio
limpieza de diálogos
DAW
herramientas de postproducción
```

## Resultado

Este bloque demuestra que la experiencia de sonido no se limita al directo. También incluye captación, edición y postproducción dentro de proyectos audiovisuales.

## Qué demuestra

```txt
Comprensión del flujo completo de audio audiovisual.
Capacidad de captación en rodaje.
Criterio para editar y limpiar diálogos.
Conexión entre sonido directo y postproducción.
Adaptación a proyectos audiovisuales y corporativos.
```

## Resumen para card

```txt
Captación de sonido en rodaje, microfonía, edición de diálogos, limpieza, diseño sonoro y mezcla para proyectos audiovisuales, cortometrajes y contenido corporativo.
```


## Qué agrupa

Este case study agrupa experiencia en soporte técnico AV para conferencias, ruedas de prensa, eventos institucionales, streaming, grabación, videoconferencia y eventos híbridos.

Incluye trabajos y contextos como:

```txt

Zoom
Microsoft Teams

```

## Contexto

El AV corporativo exige fiabilidad, comunicación clara y capacidad para resolver incidencias sin interrumpir el evento.

A diferencia de un concierto, muchas veces el foco no está en el espectáculo, sino en que la comunicación funcione: ponentes, traducción, videoconferencia, grabación, streaming, presentaciones, cámaras, audio y plataformas remotas.

## Reto

Los retos habituales incluyen:

```txt
videoconferencias críticas
eventos híbridos con público presencial y remoto
grabaciones
streaming
soporte a ponentes
integración de plataformas como Zoom o Teams
operación de sonido
gestión de incidencias rápidas
entornos institucionales o corporativos
```

## Rol asumido

```txt
Conference Technician
Corporate AV Technician
Hybrid Event Technician
Streaming Support
Sound Operator
Technical Support
```

## Enfoque técnico

El enfoque se basa en:

```txt
preparar sistemas antes del evento
verificar audio, vídeo y red
probar plataformas de videoconferencia
coordinarse con producción o responsables del evento
dar soporte a ponentes
operar sonido o streaming cuando aplica
resolver incidencias durante el evento
mantener una comunicación calmada y profesional
```

## Herramientas / sistemas

```txt
Zoom
Microsoft Teams
plataformas de videoconferencia
sistemas de sonido
microfonía
mesas de mezcla
cámaras
streaming
grabación
platos de televisión
redes AV
ordenadores de presentación
```

## Resultado

Este bloque demuestra experiencia en entornos corporativos e institucionales donde la fiabilidad técnica, la comunicación y la resolución de problemas son esenciales.

## Qué demuestra

```txt
Capacidad para trabajar en entornos profesionales exigentes.
Experiencia en videoconferencia y eventos híbridos.
Operación de sonido en contexto corporativo.
Comunicación con ponentes, clientes y equipos técnicos.
Resolución rápida de incidencias.
Comprensión de flujos AV complejos.
```

## Resumen para card

```txt

→ demuestra oficio de sonido y eventos.

Ominira
→ demuestra continuidad con una banda y experiencia en gira.

Location Sound & Postproduction
→ demuestra audio audiovisual y flujo de captación/post.

→ demuestra entornos profesionales, streaming y videoconferencia.

Homelab & Self-hosted Infrastructure
→ demuestra redes, servidores e infraestructura.

→ demuestra capacidad de desarrollo, backend, CI/CD y operación software.
```

Esta estructura cuenta mejor la historia profesional completa:

> He trabajado en eventos reales, sonido, broadcast, AV corporativo, redes e infraestructura. Y además sé construir herramientas y sistemas propios.
